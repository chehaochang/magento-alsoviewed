Magento Also Viewed Products.

Free magento module, that helps customer to find the product he is looking for. Module uses automatically collected anonymous browsing history, to suggest product alternatives.

Download For Free View Screenshots

High Perfomance.

Module code is optimized for maximum perfomance. You will not notice any changes to the site speed after installation.

Simple and Clean Design.

Extension uses standard Magento html markup and css classes, which results in great design with most themes.

Fully Responsive.

Module is provided with css media rules to make the product listing looks good when installed on the responsive theme.
